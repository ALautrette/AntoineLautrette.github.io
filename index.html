Projets

GSB

Situation : Projet réalisé en PPE de deuxième année

Objectif : Suite à une récente fusion le laboratoire Galaxy Swiss Bourdin souhaite uniformiser la gestion des frais professionnels des visiteurs médicaux via le développement en interne d’une application WEB, d'une application mobile et d'un service Windows. L'application WEB est destinée à la fois aux visiteurs médicaux et aux comptables en charges de leur gestion tandis que l'application mobile est uniquement destinée aux visiteurs. Elles permettent de saisir et d’enregistrer tous les frais engagés par les visiteurs médicaux aussi bien pour l’activité directe (déplacement, restauration et hébergement) que pour les activités annexes (événementiel, conférences, autres). Le service Windows permet d'automatiser les remboursements tout les mois à une date précise

Technologies utilisées :

  • HTML/CSS/JS, PHP et Ajax pour l'application WEB
  • Java pour l'application mobile
  • C# pour le service windows
  • MariaDB pour le SGBD
